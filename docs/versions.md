# 📌 版本说明（Changelog）

## ☕ 1.2.x 系列

---

### 🥩1.2.1
> **🔄 更新内容**

| 功能          | 更新说明                                                                                            |
|-------------|-------------------------------------------------------------------------------------------------|
| `@CDateRange` | 提供了LocalDateTime类型支持，修复了时区问题.<br/> max date自动补充最后时间 如：输入2022-06-30 判断会以 2022-06-30 23:59:59进行判断 |
| 字段allowNull    | 所有的注解都加上了判空字段allowNull 默认为true 允许为空                                                             |
---

### 🍕1.2.0


> **🆕 新增功能**

| 注解名            | 说明           |
|------------------|----------------|
| `@CBankCard`     | 银行卡校验     |
| `@CUrl`          | URL 校验       |
| `@CMoney`        | 金额校验       |
| `@CMacAddress`   | MAC 地址校验   |
| `@CIpv6`         | IPv6 校验      |

> **🔄 更新内容**

| 注解名                        | 更新说明                                        |
|----------------------------|-------------------------------------------------|
| `@CAccount` / `@CPassword` | 新增 `min`、`max` 参数，可用于控制最小/最大长度 |
| `@CDomain`                 | 域名规则增强，支持中文域名与 `-` 标识符         |
